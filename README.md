# Sahaby Restaurant

مشروع موقع مطعم متكامل باللغة العربية.

## المميزات
- عرض جميع أنواع المأكولات والمشروبات.
- صور توضيحية وأسعار لكل منتج.
- تصنيفات للوجبات والمشروبات.
- سلة طلبات مع زيادة ونقصان الكمية.
- حساب الضريبة والخصم والإجمالي.
- اختيار طريقة الدفع: كاش أو بنكك.
- تصميم Responsive يعمل على الهاتف والكمبيوتر.

## طريقة التشغيل
افتح ملف `index.html` أو ارفع المشروع على GitHub Pages.
