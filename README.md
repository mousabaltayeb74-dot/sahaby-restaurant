# Golden Bite Restaurant

مشروع موقع مطعم احترافي باللغة العربية.

## المميزات
- أكثر من 100 صنف.
- صور توضيحية للأطعمة والمشروبات.
- تصنيفات متعددة.
- بحث وفلاتر.
- سلة مشتريات.
- حساب الإجمالي والضريبة ورسوم التوصيل.
- طرق دفع: كاش وبنكك.
- إرسال الطلب عبر واتساب.
- تصميم متجاوب مع الهاتف والكمبيوتر.

## التشغيل
افتح ملف index.html أو ارفع المشروع على GitHub Pages.
